
<h1 align="center">个人博客</h1>

<p align="center">:rainbow: 基于 Laravel5.7 框架开发，后台使用 Laravel-admin，前台使用 startbootstrap-clean-blog 主题。</p>

## 安装项目

```sh
$ git clone https://github.com/Dcyer/dcynsd.git
```

```sh
$ composer install
```

```sh
$ cp .env.example .env
```

```sh
$ php artisan migrate
```
